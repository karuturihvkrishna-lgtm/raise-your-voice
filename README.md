# Raise-Your-Voice

Raise-Your-Voice is a web-based online complaint management system designed for students.

## Problem Statement
Many students hesitate to report issues such as ragging, harassment, or unfair treatment due to fear and lack of anonymity.

## Solution
This platform allows students to submit complaints online in a safe and anonymous manner, helping management identify and resolve issues effectively.

## Features
- Online complaint submission
- Anonymous reporting
- Simple and user-friendly interface
- Secure web-based platform

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages
